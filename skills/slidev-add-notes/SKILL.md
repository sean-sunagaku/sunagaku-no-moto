# Slidev プレゼンターノート同期スキル

Slidev の slides.md ファイルのプレゼンターノート（`<!-- -->` コメント）を、
各スライドに記載されているテキスト内容と完全に一致させる。

## ワークフロー

1. **対象ファイルの特定**
   - ユーザーが指定した slides.md ファイルを対象にする
   - 指定がなければ、カレントディレクトリ配下の `**/slides.md` を Glob で検索し、すべて対象にする

2. **ファイルの読み込みと解析**
   - Read ツールで slides.md を読み込む
   - `---` 区切りでスライドを分割する（frontmatter の `---` は除外）
   - 各スライドの構造を把握する

3. **各スライドのノート更新**
   - 既存の `<!-- -->` コメントがあるスライドのみ更新する（コメントがないスライドには追加しない）
   - スライド本文からテキストを抽出し、ノートの内容を置き換える
   - Edit ツールで既存コメントを新しい内容に置換する

4. **完了報告**
   - 更新したスライド数を報告する

## テキスト抽出ルール

各スライドの HTML/Markdown コンテンツからテキストを抽出する際のルール:

### 基本ルール
- HTML タグを除去し、純粋なテキスト内容のみを抽出する
- 各項目は `- ` で始まるリスト形式にする
- `<style>` タグ、`<img>` タグ、`<video>` タグの内容は無視する

### 見出しと本文の扱い
- 見出し（太字、font-weight: 600/700/800 のテキスト）→ 独立した行にする
- 見出しの下の説明テキスト → 次の行に独立して記載する

### スライドタイプ別の処理

**タイトルスライド（1枚目）:**
```
<!--
- メインタイトルのテキスト
- サブタイトルのテキスト
- 発表者名
-->
```

**セクション区切りスライド（layout: section）:**
```
<!--
- h1 タイトルのテキストのみ（<br>は除去してスペースに置換）
-->
```

**目次スライド:**
```
<!--
- 1. 項目テキスト1
- 2. 項目テキスト2
- 3. 項目テキスト3
-->
```

**コンテンツスライド（通常のスライド）:**
```
<!--
- 見出しテキスト1
- 説明テキスト1
- 見出しテキスト2
- 説明テキスト2
-->
```

**デモ動画・画像のみのスライド:**
```
<!--
- h1 タイトルのテキストのみ
-->
```

**まとめスライド:**
```
<!--
- まとめ項目1
- まとめ項目2
-->
```

**外部ファイル参照スライド（src: を含むスライド）:**
- ノートの更新対象外。スキップする。

## 具体的な変換例

### 入力（スライド本文）:
```html
<div style="padding: 1.5rem; border-left: 4px solid #4a6b8a;">
  <div style="font-size: 1.3rem; font-weight: 600; color: #4a6b8a;">見出しテキスト</div>
  <div style="font-size: 0.95rem; color: #64748b;">説明テキスト</div>
</div>
```

### 出力（プレゼンターノート）:
```
<!--
- 見出しテキスト
- 説明テキスト
-->
```

### 入力（セクションスライド）:
```markdown
---
layout: section
---

# 1. バイブコーディングは、<br>事前準備が9割
```

### 出力:
```
<!--
- 1. バイブコーディングは、事前準備が9割
-->
```

## 注意事項

- `<!-- -->` コメントが存在しないスライドには新規追加しない
- `src:` で外部ファイルを参照しているスライドはスキップする
- `<br>` タグはスペースまたは空文字に置換する
- スタイル属性（style="..."）は無視する
- class 属性は無視する
- HTML エンティティ（`&amp;` など）はデコードする
- 空行や余計な空白は除去する
